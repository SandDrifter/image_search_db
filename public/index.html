Image Search Microservice

Functionality

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example API Requests

  • https://relwiwa-image-search.herokuapp.com/search/cats?offset=10
  • https://relwiwa-image-search.herokuapp.com/history

Example JSON Output: Image Search

  • [{"imageUrl": "www.zastavki.com/eng/Animals/Cats/wallpaper-23761.htm", "hostPageUrl": "http://www.zastavki.com/eng/Animals/Cats/wallpaper-23761.htm", "altText": "Cat wallpapers and images - wallpapers, pictures, photos"}, {...}]

Example JSON Output: Search History

  • [{"searchTerm": "rabbits", "searchTime": "2016-10-05 19:35:03"}, {...}]
